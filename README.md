Simple HTML5 uploader for express

* Multiple uploads 

# REQUIREMENTS

* nodejs + npm

# DEV

## Install

CLI:

`npm install`

## Starting and Stopping

To start server with supervisor:

`nodemon app.js`

To end process: 

Press:
 CTRL + C
 Y
 ENTER

# PRODUCTION

## Install

CLI:

`npm install --production`

## Starting and Stopping

Start:

`NODE_ENV=production node app.js`

End:

Press:
 CTRL + C
 Y
 ENTER
